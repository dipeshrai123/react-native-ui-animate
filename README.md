# react-native-ui-animate

React Native library for gestures and animation

## Installation

```sh
npm install react-native-ui-animate
```

## Usage

```js
import UiAnimate from "react-native-ui-animate";

// ...

const result = await UiAnimate.multiply(3, 7);
```
